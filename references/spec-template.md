# 规格书模板 / Product Specification Template

> 产品对外规格参数表，简洁、量化、可对比。用于产品选型、招投标、数据手册。

## 封面 / Cover
- 产品名称、型号、规格书版本、日期、制造商

## 1. 产品概述 / Product Overview
- 简介、特性亮点、典型应用、外观图

## 2. 型号说明 / Model Nomenclature
- 型号命名规则（如 OBC-6K6-400-B：6.6kW / 400V / 双向）

## 4. 保护功能 / Protection
- 过压 OVP / 欠压 UVP / 过流 OCP / 过温 OTP / 短路 SCP / 反接保护
- 每项：阈值、动作、恢复方式

## 5. 通信与接口 / Communication
- CAN / LIN / 以太网 / CP（充电控制导引）
- 协议版本、波特率、诊断（UDS）

## 6. 机械规格 / Mechanical
- 尺寸 (L×W×H)、重量、安装方式、接插件型号、防护等级 IP

## 8. 可靠性 / Reliability
- 设计寿命、MTBF、质保期

## 9. 符合标准 / Compliance
- 安规、EMC、环保（RoHS/REACH）标准清单

## 10. 订购信息 / Ordering Information
- 型号对照表、选配项
