# 控制策略库 / Control Strategies Library

> 基础控制（PI/PR/前馈/解耦）+ 10 个高级模型。引用到 SOR 技术方案时指明控制策略名称与关键参数（带宽、采样率、传感器需求）。

---

## 基础控制 / Basic Control

### PI（比例积分）
- **用途**：直流量调节（电压环/电流环）、双闭环（电压外环 + 电流内环）、DC-DC 所有拓扑基础。
- **参数**：Kp、Ki；电流环带宽 = fs/5~10，电压环带宽 = 电流环/5~10。
- **数字实现**：双线性变换 / 前向欧拉；抗积分饱和（Back-calculation / Clamping）。

### PR（比例谐振，Proportional-Resonant）
- **用途**：AC 量无静差跟踪（并网逆变器电流控制、单相 PFC）。
- **传递函数**：Gc = Kp + (2·Ki·ωc·s)/(s²+2ωc·s+ω₀²)，ω₀ = 基频（50/60Hz），ωc 截止带宽。
- **优势**：直接 αβ 坐标，不需 dq 变换（单相尤其简洁）；多谐波抑制可级联多个 PR。

### 前馈解耦 / Feedforward & Decoupling
- **电压前馈**：Vo_ff = Vin·D（理想占空比），加速瞬态响应。
- **dq 解耦**：三相逆变 d 轴控制有功、q 轴控制无功/电压，交叉耦合项 ±ω·L·i 前馈抵消。
- **负载前馈**：已知负载电流 Io，减轻反馈环负担。

## 高级控制模型 / Advanced Control Models（10 个）

### 1. Totem-Pole GaN PFC（图腾柱 GaN 功率因数校正）
- **拓扑**：无桥 PFC，上下快管（GaN/SiC 650V）+ 两工频管（Si MOSFET）。
- **控制**：双闭环（母线电压外环 + 电流内环）+ 电流整形（sin 包络跟随）；CCM/CRM/DCM 三模式切换。
- **关键技术**：
  - **死区补偿**：GaN 快管导通 Qrr 几乎为零，但体二极管反向恢复时 shoot-through 风险 → 自适应死区 + 源极电感 Ls。
  - **工频管零电压切换**：工频管在电流过零附近切换（ZVS/ZCS）。
  - **EMI 抑制**：高 fs（100~500kHz）带来共模干扰 → Y 电容 + CM 扼流圈 + 软开关。
- **适用**：OBC 前级、充电桩、服务器电源，效率 >99%。
- **参数指标**：fs = 100~500kHz；输入 THD <3%；母线 Vo = 380~420V（单相）、700~800V（三相）。

### 2. MMC HVDC（模块化多电平换流器高压直流输电）
- **拓扑**：每相由 N 个半桥子模块（SM）串联，输出 2N+1 电平。
- **控制分层**：
  1. **上层调度**：有功/无功功率指令（Pref/Qref）→ dq 电流指令。
  2. **中层环流控制**：三相桥臂间环流 i_cir 抑制（PR 或 PIR）。
  3. **底层电压均衡**：每个 SM 的电容电压均衡（排序 + 轮询投切、最近电平逼近 NLM）。
- **调制**：最近电平调制 NLM / 载波移相 CPS-PWM。
- **适用**：中高压柔直（±10kV ~ ±800kV）、海上风电并网、城市电网互联。
- **参数指标**：SM 数量 N = 50~400（按电压等级），子模块电容 C_SM，环流抑制带宽 >200Hz。

### 3. PMSM FOC（永磁同步电机磁场定向控制）
- **原理**：Park 变换 abc → dq，d 轴磁链、q 轴转矩解耦；Te = (3/2)·p·ψf·iq（表贴式 PMSM id=0 控制）。
- **控制结构**：速度环（外环 PI）→ 电流环（内环 PI，dq 解耦）→ SVPWM。
- **观测器**：位置/速度传感器（Resolver/编码器）或无传感器（滑模观测器 SMO / 扩展卡尔曼 EKF / 高频注入）。
- **弱磁控制**：高速区 id<0 负 d 轴电流去磁，扩展恒功率区。
- **适用**：新能源汽车电驱 MCU（永磁同步电机/IPM）、电动摩托车驱动、工业伺服。
- **参数指标**：电流环带宽 >1kHz，速度环 50~200Hz；PWM 频率 10~20kHz（IGBT）、40~80kHz（SiC）。

### 4. VSG（Virtual Synchronous Generator，虚拟同步发电机）
- **动机**：逆变器模拟同步发电机惯性与阻尼，提供电网频率支撑（新能源渗透率高时必需）。
- **核心方程**（二阶摇摆方程）：J·(dω/dt) = Pm − Pe − D·(ω−ωn)，J 虚拟惯量，D 阻尼，ωn 额定角频率。
- **控制框架**：功率环（模拟调速器 + 摇摆）→ dq 电流环 → SVPWM；无功-电压下垂。
- **关键参数**：J（越大惯性越强，但瞬态越慢），D（阻尼系数），下垂系数 m、n。
- **适用**：储能 PCS 并网（Grid-forming 模式）、微电网主电源、光伏逆变器惯性支撑。
- **参数指标**：J = 1~10 s（虚拟转动惯量时间常数），D = 10~100，频率响应时间 <100ms。

### 5. CLLLC（双向谐振 DC-DC）
- **拓扑**：原副边对称 Cr-Lr 谐振网络 + 变压器漏感/励磁电感，两侧各一 H 桥。
- **控制**：**调频控制**（fs 调节增益 M = Vo/nVin），正向/反向能量流由频率方向决定；轻载加 **burst 模式** + PWM 混合防止增益失控。
- **软开关**：全负载 ZVS（谐振电流提前器件电压换向）；fs≈fr 时增益≈1（匹配点）。
- **设计关键**：谐振参数（Lr、Cr、Lm）匹配，品质因数 Q = √(Lr/Cr)/Rac；变压器漏感 + 外接电感 Lr。
- **适用**：OBC 双向（V2G）、储能 DC-DC、800V/400V 平台互联，3.3~22kW。
- **参数指标**：fr = 100~500kHz，调频范围 0.8fr~1.2fr；满载效率 >97%。

### 6. FCS-MPC（Finite Control Set Model Predictive Control，有限集模型预测）
- **原理**：预测 k+1 时刻所有可能的开关状态（如三相两电平逆变器 8 个矢量 V0~V7），用代价函数 J 选最优。
- **代价函数**：J = |i_d,ref − i_d(k+1)|² + |i_q,ref − i_q(k+1)|² + λ·switch_count（电流误差 + 开关频率惩罚）。
- **优势**：非线性约束直接嵌入、多目标优化、动态快。
- **挑战**：高采样率（ ≥10 倍 PWM）、变开关频率（固定 fs 需 M²PC）。
- **适用**：高性能电驱（赛车/工业）、APF 有源滤波器、快动态电源。
- **参数指标**：预测步长 Np = 1~3，控制周期 <50μs，λ 权重调优。

### 7. ADRC（Active Disturbance Rejection Control，自抗扰控制）
- **核心**：扩张状态观测器 ESO 估计"总和扰动"（参数变化+外扰+未建模），前馈补偿 → 系统变"积分器串联"→ 非线性/线性状态误差反馈 NLSEF/LSEF 控制。
- **三部件**：TD 跟踪微分器（安排过渡）、ESO、NLSEF/LSEF。
- **优势**：鲁棒性强（对电机参数/负载扰动不敏感），调参直观（带宽参数 ω₀、ωc）。
- **适用**：PMSM 电流/速度环替代 PI、PFC 电流环、飞轮储能、机器人关节。
- **参数指标**：ESO 带宽 ωo = 3~10 倍控制带宽 ωc，b₀ 补偿增益。

### 8. Triple-Phase-Shift DAB（三重移相 DAB）
- **动机**：单移相 DAB 轻载/偏压时回流功率大、电流应力高 → 三重移相（原边内移相 D1、副边内移相 D2、原副间外移相 φ）优化电流波形。
- **功率表达**（复杂，需数值优化）：P = f(D1,D2,φ,Vin,Vo,n,Llk)，目标最小化电流有效值 Irms。
- **控制**：实时求解优化（查表 LUT / 解析近似）或机器学习策略。
- **适用**：宽电压/功率范围 DAB（V2G、储能、800V 平台），效率提升 1~3%。
- **参数指标**：D1、D2 ∈ [0,0.5]，φ ∈ [0,π]；需高性能 DSP/FPGA。

### 9. Interleaved Boost PFC（交错并联 Boost 功率因数校正）
- **拓扑**：N 路 Boost 相位错开 360°/N（如 2 路错开 180°）。
- **优势**：输入电流纹波抵消 N² 倍、单管电流应力降至 1/N、磁件分散、热管理好。
- **控制**：单电压环 + N 个电流环（均流+相位管理）；平均电流法 / 峰值电流法。
- **适用**：>3kW PFC（充电桩/OBC 前级/服务器），N=2~4 常见。
- **参数指标**：相位误差 <5°，电流不平衡度 <10%，THD <5%。

### 10. STATCOM（Static Synchronous Compensator，静止同步补偿器）
- **拓扑**：电压源型逆变器（VSI，通常三相两电平/三电平）+ DC 侧电容 + AC 侧电抗器，并网但不交换有功（P≈0），纯无功调节。
- **控制**：dq 坐标，d 轴控制有功（维持 DC 电压恒定补偿损耗），q 轴控制无功（Q 指令/电压调节）。
- **快速响应**：<10ms 无功阶跃（比 SVC 电抗器快 10 倍）。
- **适用**：电网无功补偿、电压支撑、不平衡/谐波治理（加谐波电流控制）、新能源场站并网。
- **参数指标**：容量 ±1Mvar ~ ±100Mvar，响应时间 <半周波（10ms@50Hz）。

## 控制策略选型速查 / Strategy Selection

| 应用场景 | 推荐控制 | 关键要点 |
|---|---|---|
| DC-DC 恒压/恒流 | 双闭环 PI | 电压外环 + 电流内环，带宽 5~10 倍分离 |
| 单相 PFC | PR（电流） | 基频谐振无静差，50/60Hz ω₀ |
| 三相逆变并网 | dq-PI + SVPWM | 解耦 + 前馈，电流环 >1kHz |
| 电机驱动 | FOC / ADRC | FOC 成熟，ADRC 高鲁棒 |
| 微电网并网 | VSG | 提供惯性支撑，Grid-forming |
| OBC 双向 | CLLLC（调频）+ Totem-Pole PFC | 高效、双向 |
| 快动态/多目标 | FCS-MPC | 非线性约束、变频率 |
| 宽范围高效 | TPS-DAB / Burst+PFM 混合 | 优化轻载 |
| 大功率 PFC | Interleaved Boost | 纹波抵消、均流 |
| 无功补偿 | STATCOM (dq-PI) | 快速无功调节 |

## 传感器与采样需求 / Sensor Requirements
- **电压采样**：母线/相电压，精度 ±0.5%，采样率 ≥2 倍 PWM（数字控制 ADC 12~16 bit）。
- **电流采样**：相电流（霍尔/分流器），带宽 ≥开关频率，FOC 需 abc 三相或 αβ 重构。
- **位置/速度**：旋变（Resolver）/ 编码器 / 无传感器观测器（SMO/PLL）。
- **温度**：IGBT/MOSFET 结温（NTC），过温保护。
- **隔离**：高压侧采样需隔离放大器（AMC1x / ACPL-Cx）。
